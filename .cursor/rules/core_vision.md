Главная цель: создать локальную систему, которая в большинстве сценариев превосходит Cursor AI

Требования:
• 100% локальная работа по умолчанию (Ollama + ChromaDB)
• Опциональный веб-поиск через requests + BeautifulSoup только когда локальный RAG даёт низкую уверенность
• Полностью русский интерфейс в CLI и логах
• Строгий TDD-подход
• Рефлексия после каждой итерации
• Память успешных/неуспешных паттернов
• Модульная архитектура, LangGraph для workflow
• Покрытие тестами ≥ 85–90%
• Type hints + docstrings везде
